About CodeAudit Pro

CodeAudit Pro is a powerful SaaS platform designed for developers who need to ensure their code meets high standards of quality, security, and marketplace compliance.

By automating the review process, it identifies vulnerabilities, code smells, and architecture violations in seconds—saving hours of manual debugging and reducing rejection rates from quality assurance teams.

Installation Guide

Step 1: Upload & Extract

After your purchase, you will receive a main ZIP file. Extract it to find the installable.zip archive.

Upload the installable.zip to your server's public directory (or desired folder) and extract its contents.

Step 2: Database Setup

Create a database for your application. You can do this via phpMyAdmin or the command line:

CREATE DATABASE codeaudit_db; 📋 Copy

Step 3: Environment Configuration

Rename .env.example to .env and update the database credentials:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=codeaudit_db
DB_USERNAME=your_username
DB_PASSWORD=your_password

# Stripe Payment Gateway
STRIPE_KEY=your_stripe_key
STRIPE_SECRET=your_stripe_secret
STRIPE_WEBHOOK_SECRET=your_stripe_webhook_secret 📋 Copy
💳 Payment Gateway: Add your Stripe API keys as shown above to activate the subscription and billing system.

Step 4: Post-Installation & Permissions

Run the storage link command and ensure your web server has write access to the following directories:

php artisan storage:link
chmod -R 775 storage/
chmod -R 775 bootstrap/cache/ 📋 Copy

Background Jobs (Queue Worker)

⚠️ Essential Requirement: This step is mandatory for the system to function correctly.

CodeAudit Pro uses asynchronous processing for heavy tasks such as code analysis and email dispatching to ensure a smooth user experience. These tasks are added to a "Queue".

How it Works

Heavy tasks (like scanning large projects or sending emails) are handled in the background. The command below must be running to pick up these tasks. Without it, processes will remain pending.

Terminal Setup

To set this up, log into your hosting control panel or server via SSH, navigate to your project root, and execute the code in the terminal:

# Standard command (Requires terminal to stay open)
php artisan queue:work

# Background command (Stays active after closing terminal - Recommended)
nohup php artisan queue:work > /dev/null 2>&1 & 📋 Copy
🚩 Critical: Without this process running, all scans and emails will stay in "Pending" or "In Progress" status and will never complete.

If you need help implementing this on your server, please contact our support team from the link in the header.

Admin Panel: Dashboard & Reports

The Admin Dashboard provides a real-time birds-eye view of all platform activity, including scan volumes, server health, and revenue stats.

Global Audit Reports

Admins can access all reports generated by users across the platform. This centralized view allows you to monitor what's being scanned and view detailed compliance reports directly.

Managing Subscriptions & Plans

Create various pricing tiers to monetize your platform. You can balance limits based on Scan Counts or Time Duration.

Adding a New Plan

  1. Navigate to Plans Management.
  2. Click "Create Plan".
  3. Set the price, trial days, and limit values.
  4. Select the Detection Services available for this plan.

Plan Details & Pricing

Configure your plan fields as shown in the interface. Important: The price field must be entered in cents (e.g., enter 1000 for $10.00) to ensure precise transaction processing.

Extending with Plugins

CodeAudit Pro is highly modular. You can upload custom "Detection Plugins" to add new rules to the scanner engine.

How to Upload a Plugin

  1. Go to Integrations page.
  2. Click "Upload Integration".
  3. Select the .zip file of the plugin.
  4. Once installed, it will appear in the list where you can configure its API keys or toggle it ON/OFF.

User Guide: Scanning Projects

Users can upload their project archives to get a detailed breakdown of issues found.

  1. Go to Audits.
  2. Drag and drop your .zip file.
  3. Click "Start Audit".
  4. The system will queue the scan and notify you once the report is ready.
